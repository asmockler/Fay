Fay.js

An animated SVG icon library for
discerning developers

Fay is an easy-to-implement animated icon library.

Icons are added to any element by including simple class names such as .fay-menu or .fay-chevron.

Say Goodbye to Rough Edges

Fay icons are drawn with svg in the browser - this means that they will never be pixelated and are infinitely scalable.

Fay Plays Nice with Everyone

Fay was built with the wonderful Raphael.js library and with compatibility in mind. Icons will render and animate beautifully on tons of browsers.

Options, Options, Options

Fay was built to be as creative and flexible as you are. Tons of options can be adjusted right in the html for quick and flexible designs. Your icon library shouldn't lock you in.

data-fay-scale - Scale the icons to be any size. Default is 50. Number only - do not include px.

data-fay-stroke - Change the stroke width of the icon. Default is 2. Scales automatically if not declared.
